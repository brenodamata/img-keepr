## IMG Keepr

This is a simple personal image hosting app built with these technologies:

* **Ruby** *2.2.1p85* ✅

* **Rails** *4.2.5.1* ✅

* **PostgreSQL** *database* ✅

* **Heroku** *deployment* ✅

* [**Sendgrid**](https://app.sendgrid.com) *sending emails in production* ✅

* **AWS** *file storage* ❌

* Test framework ❌

* **Sidekiq** *background processing* ❌
